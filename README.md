<h2> 💡 핵심 기능 💡 </h2>

**로그인/회원가입 페이지**
+ JWT 인증 방식을 이용한 로그인/회원가입 기능

**메인 페이지**
+ 식물 검색 기능
+ 식물 리스트업 기능
+ 리스트업 내 상세 페이지 이동 기능
+ 로그아웃 기능

**상세 페이지**
+ 식물 설명 렌더링
+ 식물별 꿀팁 리스트업
  : 꿀팁 등록
  : 꿀팁 삭제
